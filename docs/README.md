## [컨트롤러] 예외처리하는 클래스 ExceptionHandling 정의
- 입력받은 코치의 이름 글자수가 2~4자가 아닐 시 에러
- 입력받은 코치의 이름에 중복이 있을 시 에러
- 입력받은 코치 수가 2~5명이 아닐 시 에러
- 못 먹는 메뉴가 0~2개가 아닐 시 에러
- 못 먹는 메뉴가 실제 메뉴판에 없는 음식일 시 에러
